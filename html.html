Introducción a HTML

HTML (Hypertext Markup Language) es un lenguaje de marcado utilizado para crear páginas web. Te proporciona la estructura básica y el contenido de una página web, permitiéndote organizar y presentar información de manera jerárquica y semántica.

Etiquetas HTML

En HTML, una etiqueta es una pieza fundamental que se utiliza para marcar o etiquetar diferentes partes de un documento. Cada etiqueta tiene un propósito específico y define cómo se debe mostrar o interpretar el contenido que rodea. Existen dos tipos principales de etiquetas en HTML:

  • Etiquetas de Auto-cierre:
  • Son etiquetas que no tienen una etiqueta de cierre correspondiente. Se utilizan para elementos que no contienen contenido, como imágenes o saltos de línea. Por ejemplo:

Las etiquetas también pueden tener atributos, que proporcionan información adicional sobre el elemento. Los atributos se colocan dentro de la etiqueta de apertura y se utilizan para configurar propiedades específicas del elemento. En el siguiente ejemplo:

  • < a > : es la etiqueta de enlace.
  • href: es un atributo que especifica el destino del enlace.

También es importante saber que las etiquetas pueden anidarse, lo que significa que puedes colocar una etiqueta dentro de otra. En este caso, la etiqueta interna debe ce- rrarse antes de la etiqueta externa

En este caso vemos que la etiqueta head tiene como contenido a otra etiqueta title, por lo que:

  • La etiqueta head es el padre de la etiqueta title y meta.
  • La etiqueta title y meta son hijas de la etiqueta head.
  • La etiqueta title y meta son hermanas.
Si nos fijamos, además, la etiqueta anidada (title) está en una nueva línea y con un nivel de indentación/sangrado mayor. Esto es así por una convención mundial a la que se ha llegado para que los programadores escribamos código de una manera similar, tanto para hacernos más fácil y comprensible el código cuando este crezca, como para cuando tengamos que compartirlo con otros programadores

Estructura básica de una página web

Una página HTML sigue una estructura estándar que incluye varios elementos esencia- les para definir el contenido y la configuración de la página.


A continuación explicamos la función que cumple cada etiqueta:

  • < !DOCTYPE html>: Esta es la primera línea de cualquier documento HTML. In- dica al navegador qué versión de HTML se está utilizando. En este caso, HTML es la versión 5, que es la más utilizada en la actualidad.
  • < html>:Es el elemento raíz de un documento HTML. Contiene todo el contenido de la página web. Todas las etiquetas deben estar incluidas dentro.
  • < head>: Proporciona información sobre el documento, pero no contiene conte- nido visible en la página. En su lugar, contiene metadatos, enlaces a archivos ex- ternos (como CSS), enlaces a fuentes, y otros elementos que no se muestran di- rectamente en la página.
  • < meta charset=ÜTF-8>: Especifica el conjunto de caracteres (UTF-8) que se uti- lizará para interpretar el contenido. Esto asegura que los caracteres especiales y acentos se muestren correctamente.
  • < title>:Define el título de la página que se muestra en la pestaña del navegador
  • < body>: Es donde se coloca todo el contenido visible de la página web, como tex- to, imágenes, enlaces y otros elementos.
