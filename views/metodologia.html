¿CÓMO LO VAMOS A APLICAR EN EL AULA?
Distribución Aula Invertida Puzzle de Aronson
Explicación resumen de las características de cada metodología
Semana 1 - El profesor prepara ese contenido y lo publica en el aula virtual del grupo.
- El alumno estudia teoría en casa mediante videos y lecturas.
- En clase se resuelven dudas y se trabaja el proyecto grupal.
- Enfoque práctico desde el inicio. 		- Aprendizaje cooperativo.
- Cada alumno investiga una parte del contenido.
- Luego, enseña a su grupo y aplican todo al proyecto conjunto.
- Refuerza la responsabilidad y la comprensión global.
Roles del puzle de Aronson:
- Tutor: guía el aprendizaje dentro del grupo.
- Coordinador: organiza tareas y tiempos.
- Evaluador: verifica la calidad del trabajo y comprensión de todos. - Documentador: Toma nota de los acuerdos, recopila recursos y se encarga del material de entrega.
Programación de la práctica
Semana 2 Estudiar estructura básica HTML que se ha subido al aula virtual
En clase: se inicia el diseño de la web del grupo con estructura HTML. 		Cada alumno del grupo se especializa en una parte del HTML:
- Estructura y etiquetas básicas
- Cabecera y pie de página
- Navegación
- Secciones de contenido
Semana 3 Estudiar JavaScript básico para aplicarlo a la web HTML
En clase: añaden interactividad a su página. 		Cada alumno del grupo se especializa en una parte del JS:
- Variables y tipos
- Funciones
- Eventos
- Integración JS en HTML
Semana 4 Estudiar como se aplican los estilos con CSS
En clase: aplicarán estos estilos al HTML del proyecto. 		Cada alumno del grupo se especializa en una parte del CSS:
- Estilos base y tipografía
- Colores y fondos
- Identificadores y clases
- Organización del CSS
Semana 5 Reunión de expertos
Semana 6 Estudiar elementos básicos de maquetación
En clase: maquetan secciones principales de su web. 		Cada alumno del grupo se especializa en una maquetación del CSS:
- Box model
- Flexbox
- Grid
- Espaciado y alineación de elementos
Semana 7 Estudiar como añadir elementos externos.
En clase: añadirán a la web enlaces e imágenes externas. 		Cada alumno del grupo añadirá un tipo de enlace diferente:
- Imágenes
- Páginas externas
- Videos youtube
- Interacción con formularios
Semana 8 Estudiar fundamentos de usabilidad y accesibilidad web.
En clase: mejoran la experiencia de uso de su web. 		Cada alumno del grupo aportará una opción de mejora de la interfaz de usuario:
- Accesibilidad
- Navegación clara
- Evaluación heurística del sitio
Semana 9 Segunda reunión de expertos
Semana 10 Revisión final y ajustes de la web del grupo.
Autoevaluación y exposición del proyecto.
Uso de rúbricas. 		Cada grupo revisa el trabajo conjunto, corrige errores y se prepara para explicar lo aprendido y su aplicación. Los roles rotan en la presentación.
