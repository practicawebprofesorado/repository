Introducción a CSS

CSS, siglas de Cascading Style Sheets o Hojas de Estilo en Cascada, es un lenguaje fundamental en el desarrollo web que se utiliza para definir la apariencia visual de los documentos HTML. Gracias a CSS, los desarrolladores pueden controlar cómo se presentan los elementos en una página web, incluyendo aspectos como colores, tamaños, tipos de letra, márgenes, espaciado, alineación y disposición de los contenidos.


Este lenguaje permite separar el contenido (HTML) del diseño, lo que facilita la creación de sitios web más limpios, organizados y fáciles de mantener. CSS también permite aplicar estilos de forma consistente en múltiples páginas, mejorar la accesibilidad y optimizar la experiencia del usuario. Además, ofrece herramientas para adaptar el diseño a distintos dispositivos, como móviles, tabletas y pantallas de escritorio, mediante técnicas como el diseño responsivo (responsive design).



Cómo enlazar CSS con HTML

Para aplicar estilos a una página web, es necesario vincular tu archivo CSS al documento HTML. Esto se logra utilizando la etiqueta <link> dentro de la sección <head> del archivo HTML. Este enlace le indica al navegador que debe cargar y aplicar los estilos definidos en el archivo CSS externo.


Los pasos para enlazar CSS a HTML serían:

  • Crea un archivo CSS: escribe tus reglas de estilo en un archivo con extensión .css, por ejemplo, styles.css.
  • Guarda el archivo: puedes colocarlo en la misma carpeta que tu HTML o en una subcarpeta como css/. Asegúrate de conocer la ruta exacta.
  • Agrega el enlace en HTML: dentro de la etiqueta <head> de tu documento HTML, incluye la siguiente línea:
  • rel=“stylesheet” indica que el archivo enlazado es una hoja de estilos.
  • href=“...” especifica la ubicación del archivo CSS.
Cuando el navegador carga la página, también buscará el archivo CSS y aplicará automáticamente los estilos definidos. Esta separación entre estructura (HTML) y presentación (CSS) mejora la organización del código, facilita el mantenimiento y permite reutilizar estilos en múltiples páginas.


Sintaxis de una regla CSS

Antes de comenzar a aplicar estilos, es fundamental comprender la estructura básica de una regla CSS. Cada regla está compuesta por tres partes principales: el selector, la propiedad y el valor. La sintaxis general es la siguiente:

  • Selector: indica a qué elemento(s) HTML se aplicará la regla. Puede ser una etiqueta, una clase, un ID o una combinación más compleja.
  • Propiedad: es la característica visual que deseas modificar, como el color, el tamaño de fuente, el margen, etc.
  • Valor: especifica el ajuste que se aplicará a la propiedad, como red, 16px, center, entre otros.
Por ejemplo, la regla p { color: blue; } aplicará el color azul a todos los párrafos (<p>) del documento. Esta estructura clara y flexible permite personalizar el diseño de una página web con precisión y eficiencia. Además, puedes agrupar múltiples propiedades dentro de un mismo selector para aplicar varios estilos a la vez.


Selección de elementos en CSS

La selección de elementos es uno de los aspectos más importantes en CSS, ya que permite aplicar estilos de forma precisa a los elementos HTML de una página web. CSS ofrece una amplia variedad de selectores que permiten apuntar a elementos individuales, grupos de elementos o incluso a partes específicas de un elemento.


Los tipos de selectores más comunes son:

  • Selector por etiqueta: aplica estilos a todos los elementos de un tipo específico. Este ejemplo cambia el color del texto de todos los párrafos <p> a rojo.
  • Selector por clase: aplica estilos a todos los elementos que comparten una clase. Útil para aplicar estilos a múltiples elementos sin importar su tipo
  • Selector por ID: aplica estilos a un único elemento con un identificador único.
  • Combinación de selectores: permite seleccionar elementos dentro de otros elementos. Este ejemplo aplica el color verde solo a los párrafos dentro de un elemento con la clase contenedor.
  • Pseudo-clases y pseudo-elementos: permiten aplicar estilos en situaciones específicas o a partes de un elemento. Aquí, el fondo del enlace cambia cuando el usuario pasa el cursor sobre él.
La selección de elementos es una parte fundamental en CSS, ya que permite aplicar estilos de forma precisa y controlada a los distintos componentes de una página web. Gracias a la variedad de selectores disponibles, es posible dirigir estilos tanto a elementos individuales como a grupos completos, según su tipo, clase, ID o posición dentro del documento HTML. En el siguiente apartado, profundizaremos en las propiedades y valores más comunes que se utilizan para definir estilos.

Cascada y especificidad en CSS

Uno de los conceptos clave para entender cómo se aplican los estilos en CSS es la cascada. Este término hace referencia al proceso mediante el cual el navegador decide qué reglas de estilo aplicar cuando múltiples declaraciones afectan al mismo elemento y propiedad. CSS evalúa estas reglas siguiendo un orden de prioridad basado en tres factores principales: especificidad, orden e importancia.

  • Especificidad

    • La especificidad determina qué regla tiene más peso cuando varias compiten por el mismo elemento. Se calcula en función del tipo de selectores utilizados:

    • Selectores de ID (#id) tienen más peso que clases (.clase) o etiquetas (div, p).
    • Cuanto más específico sea el selector, mayor será su prioridad.
    • Por ejemplo, #menu .item tiene más especificidad que .item.
  • Orden de aparición

    • Si dos reglas tienen la misma especificidad, se aplicará la que aparezca más abajo en el archivo CSS, ya que CSS lee de arriba hacia abajo.


  • Importancia

    • La declaración !important puede forzar la aplicación de un estilo, incluso si otra regla tiene mayor especificidad. Sin embargo, debe usarse con precaución, ya que puede dificultar el mantenimiento del código.

Comprender cómo funciona la cascada y cómo se calcula la especificidad es esencial para escribir estilos eficientes y evitar conflictos inesperados.


El Modelo de caja en CSS (Box Model)

El Box Model o Modelo de Caja es un concepto esencial en CSS que describe cómo se estructura visualmente cada elemento HTML en una página web. En CSS, todos los elementos se representan como cajas rectangulares que están compuestas por diferentes capas que rodean el contenido.



Componentes del Box Model:

  • Contenido (Content): es el área donde se muestra el contenido del elemento, como texto, imágenes o formularios.
  • Relleno (Padding): es el espacio entre el contenido y el borde del elemento. Se puede ajustar individualmente con propiedades como padding-top, padding-right, padding-bottom y padding-left.
  • Borde (Border): rodea el contenido y el padding. Puede tener diferentes grosores, estilos y colores mediante propiedades como border-width, border-style y border-color.
  • Margen (Margin): Es el espacio exterior que separa el elemento de otros elementos cercanos. También se puede configurar por lados con margin-top, margin-right, margin-bottom y margin-left.

Comprender el modelo de caja es fundamental para controlar el diseño y la distribución de los elementos en una página web. Permite ajustar el espaciado, alinear elementos y evitar solapamientos no deseados. Además, es clave para crear diseños responsivos y bien estructurados.


¿Qué es el modelo Flexbox en CSS?

El Modelo Flexbox (abreviatura de Flexible Box) es una poderosa técnica de diseño introducida en CSS3 que permite distribuir y alinear elementos de forma eficiente dentro de un contenedor, incluso cuando su tamaño es dinámico o desconocido. Su principal objetivo es facilitar la creación de diseños flexibles, adaptables y responsivos, superando las limitaciones de los métodos tradicionales basados en flotado (float) o posicionamiento absoluto.


Flexbox organiza los elementos en una dirección principal (horizontal o vertical) y permite controlar su alineación, distribución del espacio, orden visual y comportamiento en diferentes tamaños de pantalla. Esto lo convierte en una herramienta ideal para construir interfaces modernas, como menús de navegación, galerías de productos, tarjetas, formularios y mucho más.


Gracias a propiedades como display: flex, justify-content, align-items y flex-wrap, los desarrolladores pueden lograr diseños complejos con menos código y mayor control visual.


Imágenes y fondos en CSS

CSS ofrece múltiples formas de trabajar con imágenes, ya sea como contenido visual o como parte del diseño de fondo de un elemento. Estas herramientas permiten crear interfaces más atractivas, dinámicas y personalizadas.


Imágenes como fondo

Puedes usar imágenes como fondo de cualquier elemento HTML utilizando la propiedad background-image. Por ejemplo:

  • background-size: cover hace que la imagen cubra todo el contenedor.
  • background-position: center center centra la imagen tanto horizontal como verticalmente.

Propiedades para imágenes

  • object-fit: controla cómo se ajusta una imagen dentro de su contenedor. Valores comunes: cover, contain, fill.
  • width y height: definen el tamaño de la imagen en unidades como px, %, em, etc.

Fondos en CSS

  • background-color: establece un color de fondo.
  • background-repeat: controla si la imagen se repite.
  • background-attachment: define si el fondo se desplaza con el contenido o permanece fijo.

Tipografía y colores en CSS

La tipografía y los colores son elementos fundamentales en el diseño web, ya que influyen directamente en la estética, la legibilidad y la experiencia del usuario. CSS proporciona un conjunto de propiedades que permiten personalizar el texto y su presentación de forma precisa y adaptable.


Configuración de la Tipografía

  • Familias de fuentes: puedes definir una lista de fuentes para tus elementos. Si la primera no está disponible en el sistema del usuario, se usará la siguiente.
  • Tamaño de fuente: controla el tamaño del texto con unidades como px, em, % o rem.
  • Peso de la fuente: define el grosor del texto.
  • Estilo de la fuente: aplica estilos como cursiva o inclinada.
  • Unidades relativas: usar em o rem permite que el diseño se adapte mejor a diferentes dispositivos.
  • Importar fuentes externas: puedes usar servicios como Google Fonts para mejorar la estética tipográfica.
