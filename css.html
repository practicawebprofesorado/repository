Introducción a CSS

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo que se utiliza para controlar la presentación y el diseño de un documento HTML. Permite definir cómo se ven los elementos HTML en una página web, como el color, tamaño, tipo de fuente, márgenes, espaciado y posición de los elementos. CSS trabaja en conjunto con HTML para definir el aspecto y diseño de la página web. Se utiliza para controlar el color, la tipografía, el espaciado y otros estilos visuales.

Enlazando CSS y HTML

Para incluir un archivo CSS en un documento HTML, se utiliza la etiqueta <link> en la sección <head> del documento HTML. Aquí temuestro cómo se hace:

  • Crea un archivo con extensión .css que contenga tus estilos. Por ejemplo, puedes nombrarlo styles.css.
  • Guarda tu archivo CSS en la misma carpeta que tu documento HTML o en una carpeta separada, si lo prefieres. Anota la ruta del archivo CSS.
  • Dentro del documento HTML, agrega la siguiente línea en la sección <head>

  • rel=“stylesheet”: Indica que el enlace es para una hoja de estilos.
  • href=“ruta/del/archivo/styles.css”: Especifica la ruta del archivo CSS que deseas incluir.
Cuando el navegador carga la página HTML, también buscará y aplicará los estilos definidos en el archivo CSS especificado en el enlace.

Sintaxis de una Regla CSS

Antes de profundizar en las propiedades y valores, es importante entender la estructura básica de una regla CSS:

  • Selector: Indica a qué elemento o conjunto de elementos se aplica la regla.
  • Propiedad: Es la característica que queremos modificar.
  • Valor: Es la opción que asignamos a la propiedad.

Selección de Elementos

En CSS, la selección de elementos es una parte esencial para aplicar estilos a elementos específicos en una página web. Existen varios tipos de selectores que permiten apuntar a diferentes elementos HTML. Aquí temostraré algunos ejemplos:

  • Selección por Etiqueta: Los selectores por etiqueta se utilizan para aplicar estilos a todos los elementos de un tipo específico. Por ejemplo, si queremos aplicar un estilo a todos los párrafos < p > en una página, podemos usar este selector.
  • Selectores por Clase: Los selectores por clase permiten aplicar estilos a elementos que tienen un atributo class específico. Esto es útil cuando queremos aplicar un estilo a un grupo de elementos.
  • Selectores por ID: Los selectores por ID se utilizan para aplicar estilos a un elemento específico que tiene un atributo id único.

Cascada y Especificidad en CSS

La cascada en CSS se refiere al proceso de determinar qué estilos se aplican a un elemento cuando múltiples reglas compiten por afectar elmismo atributo. La cascada sigue un conjunto de reglas que determina qué estilo tiene prioridad y se aplica al elemento.

  • Especificidad: La especificidad de una regla determina qué estilo tiene prioridad cuando múltiples reglas afectan almismo elemento.Una regla más específica prevalecerá sobre una reglamenos específica.
  • Orden: Si dos reglas tienen lamisma especificidad, la regla que aparecemás tarde en el archivo CSS prevalecerá.
  • Importancia: Algunos estilos pueden sermarcados comoimportantes (!important), lo que les da la máxima prioridad, incluso sobre reglas más específicas.

La especificidad en CSS se refiere a lamanera en que se determina cuál regla se aplicará cuando hay conflictos. Se calcula en función de ciertos selectores y su combinación en una regla.

Box Model en CSS

ElModelo de Caja es un concepto fundamental en CSS que describe cómo se representa y se organiza cada elemento en una páginaweb. Cada elementoHTML se considera una “caja” que incluye contenido, relleno (padding), borde (border) y margen (margin).

Los componentes del BoxModel son:

  • Contenido (Content): Es el área que contiene el contenido del elemento, como texto, imágenes, etc.
  • Relleno (Padding): Es el espacio entre el contenido y el borde del elemento. Se puede establecer usando propiedades como “padding-top”, “padding-right”, “paddingbottom” y “padding-left”.
  • Borde (Border): Es una línea que rodea el elemento, separando el contenido del relleno. Se puede definir con propiedades como “border-width”, “border-style” y “border-color”.
  • Margen (Margin): Es el espacio entre el borde del elemento y los elementos adyacentes. Se puede especificar con propiedades como “margin-top”, “margin-right”, “margin-bottom” y “margin-left”.

El Modelo FlexBox

El modelo Flexbox (también conocido como Flexible Box o caja fexible) es una técnica de diseño en CSS que proporciona una forma más eficiente y fexible de organizar y distribuir elementos en una página web. Fue introducido en CSS3 para abordar la dificultad de crear diseños complejos y adaptables utilizando métodos tradicionales.

Imágenes y Fondo en CSS

  • Imágenes como Fondos: Las imágenes también se pueden utilizar como fondos en elementos HTML utilizando propiedades de CSS. Por ejemplo:
  • Propiedades de las Imágenes:
    • object-fit:Controla cómo una imagen debe rellenar su contenedor manteniendo sus proporciones. Algunos valores comunes son “cover”, “contain” y “fill”.

    • background-color:Establece el color de fondo de un elemento.
    • background-repeat:Controla si una imagen de fondo se repite en el contenedor.
    • background-attachment:Determina si el fondo se desplaza con el contenido o se mantiene fijo
  • Fondos conGradiente: CSS también permite crear fondos con gradientes de color en lugar de imágenes.
